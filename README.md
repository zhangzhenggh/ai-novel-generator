## 📖 项目简介

AI小说生成器 4.5 是一款功能强大的AI辅助小说创作工具，集成了智能连贯性系统、统一评估系统、多API提供商支持等核心功能，帮助作者高效创作高质量小说。有任何问题请加QQ群进行反馈，由于前期比较赶工，有好多没完成位的一些小Bug，现在正在努力修复，加入交流群进行反馈，需要哪些功能也进行反馈QQ群号：253193620

### 核心亮点

- 🎯 **统一评估系统**：整合AI去味和质量评估，提供详细的评估报告和修改建议
- 🧠 **智能连贯性系统**：自动跟踪角色、剧情、世界观，确保小说前后一致
- 🎨 **AI去味功能**：三级去味等级（禁用/基础/强力），有效去除AI生成痕迹
- 📊 **质量评估**：多维度评估（连贯性、情节、角色、风格等），自动判断是否需要重写
- 🔌 **22+ API提供商**：支持OpenAI、Claude、Gemini、通义千问、文心一言等主流API
- 📝 **灵活提示词管理**：可视化提示词编辑器，支持模板变量和版本管理
- 💾 **项目管理**：完整的项目管理系统，支持暂停/恢复生成
- 🚀 **一键打包**：支持打包成exe可执行文件，无需Python环境即可运行

## 📋 API配置

### 步骤1：选择API提供商

1. 进入「系统设置」→「API配置」
2. 选择API提供商（如：OpenAI、通义千问等）
3. 输入API密钥
4. 选择模型
5. 点击「测试连接」验证

### 步骤2：配置生成参数

在「系统设置」→「生成参数配置」中设置：

- **Max Tokens**：单次生成最大token数（推荐：4000-8000）
- **Temperature**：创造性程度（推荐：0.7-0.9）
- **上下文模式**：
  - 摘要模式：使用摘要+前文尾部（节省token）
  - 全文模式：使用完整章节（质量更高）
  - 关闭：不使用上下文

## 📝 创建小说项目

### 方式一：自动生成

1. 进入「完整小说自动生成」
2. 填写基本信息：
   - 小说标题
   - 小说类型
   - 章节数量
   - 角色设定
   - 世界观设定
   - 剧情构思
3. 配置质量控制：
   - AI去味等级：基础去味（推荐）
   - 启用质量评估：✓
   - 最低质量分：70
   - 重写阈值：60
4. 点击「创建新项目并开始生成」

### 方式二：单章创作

1. 进入「小说创作」→「单章创作」
2. 选择或创建项目
3. 输入章节大纲
4. 点击「生成章节」

## 🐛 常见问题

### 1. API连接失败

**问题**：提示"API连接失败"

**解决方案**：
- 检查API密钥是否正确
- 检查网络连接
- 检查API提供商是否可用
- 尝试使用代理

### 2. 生成速度慢

**问题**：生成速度很慢

**解决方案**：
- 使用摘要模式代替全文模式
- 减少上下文章节数
- 选择更快的模型
- 检查网络延迟

### 3. 内存占用高

**问题**：程序占用内存过高

**解决方案**：
- 减少章节数量
- 使用摘要模式
- 定期清理缓存
- 重启程序

### 4. 生成质量差

**问题**：生成的内容质量不好

**解决方案**：
- 启用强力去味
- 启用质量评估
- 设置更高的最低质量分
- 启用自动重写
- 优化提示词

### 5. 连贯性问题

**问题**：生成的内容前后不一致

**解决方案**：
- 启用连贯性系统
- 使用全文模式
- 增加上下文章节数
- 检查角色和剧情设定
